# 사내 전용 무제한 AI PPT 번역 서버 (Docker 무중단 배포용)

본 도구는 사내 팀원 누구나 드래그 앤 드롭으로 **무제한/무료**로 한국어 PPT(회사 소개서 등)를 **서식/레이아웃 100% 유지(표, 그룹 번역 포함)**하여 글로벌 애널리스트 톤으로 번역할 수 있는 자체 웹 서버 프로젝트입니다.

> **서버의 차별성(스펙)**
>
> - **API Key 불필요 (비용 0원/안전성 확보)**: 직원 누군가의 개인 카드로 연동될 구글/OpenAI 쿼터 API에 의지하지 않고, 파이썬 전용 오픈소스(`deep-translator`) 백엔드를 10개 병렬 스레드로 돌려서 동접자가 많아져도 과금 폭탄 위험률 철벽 차단
> - **1:1 용어집 매핑 방벽(Terminology Injection)**: '버블콘' -> 'Bubblecon' 등 사전에 `edtech_glossary.json`에 정의된 모든 단어를 번역기 파이프라인 전으로 돌아가 틈새 없이 강제 주입해 고품질 애널리스트 톤 보장
> - **Zero-Downtime 인프라**: `docker-compose`의 `restart: always` 정책으로 사내 리눅스 등 서버 호스트 PC가 꺼지기 전까지 1년 365일 안 꺼지고 백그라운드에 구동 유지

---

## ⚙️ 사내 서버 배포 방법 (관리자용)

특정 서버 전용 컴퓨터에 **Docker**와 **Docker Compose**만 깔려 있으면 딱 1줄 명령어로 무중단 배포 끝입니다! 소스 코드 빌드도 도커가 알아서 합니다.

```bash
# 이 저장소를 사내 서버로 다운로드/클론 한 뒤,
cd gartner_eng

# 백그라운드 서버 무중단 구동 1줄 시작!
docker-compose up -d --build
```

이제 이 서버가 있는 컴퓨터가 아니라, 사내 직원 아무나 본인 자리의 웹 브라우저를 켜고 `http://[이서버의고정IP]:8501`에 들어오면 즉시 사용할 수 있습니다!

*(참고: 에듀테크 용어집을 추가하시려면 서버를 내릴 필요 없이 호스트의 `edtech_glossary.json` 파일만 수정하시면 `volumes` 바인딩 덕분에 바로 다음 번역 시 적용됩니다)*
